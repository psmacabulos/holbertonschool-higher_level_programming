# Python - Classes and Objects

This project covers foundational Object-Oriented Programming concepts in Python.

## Covered Topics
- What is OOP
- Classes and objects
- Private attributes
- Validation
- Methods
- Area computation

## Usage

Make files executable:
    chmod +x *.py

Run individual tests:
    python3 main_0.py
    python3 main_1.py
    python3 main_2.py
    python3 main_3.py
