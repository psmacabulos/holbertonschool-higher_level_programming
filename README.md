# Holberton School -- Higher Level Programming

This repository contains projects and exercises for the **Higher Level
Programming** curriculum at **Holberton School**.

The focus of this repository is learning and applying programming
concepts using **Python**, including debugging, documentation, error
handling, and basic game logic.

------------------------------------------------------------------------

## Contents

This repository includes tasks related to:

-   Python basics and syntax
-   Debugging Python programs
-   Writing documentation using docstrings
-   Handling user input and errors
-   Implementing simple logic-based games

Each directory corresponds to a specific project or topic assigned
during the program.

------------------------------------------------------------------------

## Requirements

-   Python 3.x
-   Ubuntu/Linux environment (local machine or container)

------------------------------------------------------------------------

## Usage

To run a Python script:

``` bash
python3 script_name.py
```

Or, if the script is executable:

``` bash
./script_name.py
```
