# JavaScript DOM Manipulation

## 📌 Overview

This project focuses on understanding and applying **JavaScript DOM manipulation** techniques to dynamically interact with web pages without reloading them.

The DOM (Document Object Model) represents the structure of an HTML document as a tree of objects, allowing JavaScript to access and modify content, structure, and styles in real time.

---

## 🎯 Learning Objectives

By the end of this project, you should be able to:

- Select HTML elements using JavaScript
- Understand differences between:
  - ID selectors
  - Class selectors
  - Tag selectors
- Modify element styles dynamically
- Read and update element content
- Create, append, and remove DOM elements
- Handle user interactions via events
- Perform HTTP requests using:
  - `XMLHttpRequest`
  - `Fetch API`
- Update the UI dynamically without page reloads
- Debug JavaScript using browser developer tools
