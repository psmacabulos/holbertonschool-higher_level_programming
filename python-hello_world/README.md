# Python - Hello, World

This directory contains introductory Python exercises for the **Hello,
World** project at **Holberton School**.

The goal of this project is to become familiar with: - Running Python
scripts - Using the Python interpreter - Printing output - Understanding
basic syntax and structure

------------------------------------------------------------------------

## Files

Each file in this directory focuses on a specific introductory concept,
such as:

-   Printing text to standard output
-   Using variables and basic expressions
-   Writing and executing Python scripts

------------------------------------------------------------------------

## Requirements

-   Python 3.x
-   Ubuntu/Linux environment

------------------------------------------------------------------------

## Usage

To run a Python script:

``` bash
python3 filename.py
```

Or, if executable:

``` bash
./filename.py
```
